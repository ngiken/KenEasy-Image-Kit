    说明

    • 处理在你的浏览器内存里完成，图片不会上传到任何服务器。
    • 最长边只会等比缩小，不会把小图放大。设 0 表示保持原尺寸。
    • 质量滑块只对 WebP 与 JPEG 生效；PNG 为无损，忽略质量。
    • 重编码会移除 EXIF（含拍摄信息与方向标记）。若原图带旋转 EXIF，浏览器会先按方向绘制。
    • 依赖库已放在 web/vendor/，可用本地静态服务完全离线使用（推荐 python -m http.server）。
